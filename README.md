# ForoHub API

API REST desarrollada con Spring Boot.

## Tecnologías

- Java 17
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate
- MySQL

## Funcionalidades

- CRUD de tópicos
- Autenticación con Spring Security
- Generación de tokens JWT
- Endpoints protegidos

## Endpoints principales

POST /login  
GET /topicos  
GET /topicos/{id}  
PUT /topicos/{id}  
DELETE /topicos/{id}
